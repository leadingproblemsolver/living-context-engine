## Why engines were suppressed

DriftGuard has one validated architectural wedge but no observed external adoption data in this repository. Building targeting systems, predictive models, team dashboards, billing, or autonomous integrations now would increase surface area without proving more user value.

## Applied operating laws

- User-owned constraints are the authority.
- Deterministic rules govern verdict precedence.
- AI interprets semantic ambiguity only.
- Missing proof cannot become Pass.
- The system does not claim passive monitoring without an integration trigger.
- Every cloud evaluation stores the exact active constraint snapshot.
- Core operational events exclude raw work and evidence.
- Technical deployment readiness is separate from live production verification.
